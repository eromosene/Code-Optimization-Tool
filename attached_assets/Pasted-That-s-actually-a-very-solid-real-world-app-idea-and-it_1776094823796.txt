🧠 What You’re Building (Simplified)

Flow:
	1.	Teacher uploads a template OMR sheet (correct answers)
	2.	Students’ OMR sheets are uploaded (or snapped via camera)
	3.	App:
	•	Detects filled circles

🏗️ How the OMR Detection Works (Core Logic)

This is the brain of your app 👇

Step 1: Preprocess Image
	•	Convert to grayscale
	•	Apply threshold (black & white)

Step 2: Detect Sheet
	•	Find edges
	•	Warp perspective (so it’s straight)

Step 3: Detect Bubbles
	•	Find circles (contours)
	•	Group them per question

Step 4: Detect Filled Option
	•	Count dark pixels inside each circle
	•	The darkest one = selected answer

⸻

🧪 Simple Example (Concept)

For each question:

Question 1:
A -> 10 pixels
B -> 80 pixels  ✅ (selected)
C -> 5 pixels
D -> 3 pixels

So answer = B

⚠️ Real Challenges (Don’t Ignore These)

Be aware of these early:

1. Lighting Conditions
	•	Shadows can break detection

2. Skewed Images
	•	Need perspective correction

3. Different OMR Designs
	•	You may need fixed template format

👉 Solution: Use standardized OMR sheets

💡 Smart Shortcut (Very Important)

👉 Use fixed grid coordinates
